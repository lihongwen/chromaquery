这是一个测试文本文件。

本文件用于测试ChromaDB Web Manager的多格式文件解析功能。

第一段：介绍
ChromaDB Web Manager是一个用于管理ChromaDB集合的Web界面，支持中文集合名称和多种文档格式的上传。

第二段：功能特性
- 支持多种文档格式：文本、PDF、Word、PowerPoint、Markdown、RTF、Excel、CSV
- 智能文档分块：递归分块、固定大小分块、语义分块
- 表格智能分析：自动识别元数据列和内容列
- 向量化存储：使用阿里云嵌入模型生成1024维向量

第三段：技术架构
前端采用React + TypeScript + Ant Design构建现代化界面，后端使用FastAPI + ChromaDB提供高性能API服务。

第四段：使用场景
适用于知识库管理、文档检索、智能问答等场景，特别是需要处理中文文档的企业应用。

这是文档的结尾部分，用于测试分块算法的效果。
